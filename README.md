Pokemon TCG Archive 1999 PHP

In this project you get access to a website designed with php which contains the first three sets of the pokemon trading card game of 1999.
The Card sets can be viewed at individual pages as well as being added to private collections to account for which cards the User already in his collection.


The website has been created with phpstorm and is being displayed through a localhost with XAMPP v3.3.0
It utilizes Apache and MySQL

First install and activate the XAMPP v3.3.0 Control Panel
Start the Apache and MysQL Modules

Then to open the stage copy this adress into your browser: http://localhost/php_project/Codebase/login.php

It takes you to the greeting page where you can create an account or log in with existing user data.
When creating a profile your userdata gets saved into the SQL Database with a hashed password.

After logging in you can navigate the website 

HOMEPAGE
You can read a small exert about the history of the TCG and how it started in 1999

Collection
Here you can view and create your own collections. Collections must be created here before any given card can be added into one.

Profile
Here you can see and edit all your userdata connected to your profile

Other Sets
Here you can select your set of choice out of the three original sets from 1999 containing 150 pokemon from Gen 1
Its separated into three sets
Baseset 1999
Jungle 1999
Fossil 1999

On these specific pages you'll encounter the cards within agrid structure and can hover over them to slightly enlarge them, or click onto them to completely staticly select them for better view.
Each card shows the following info beanth its card art: 
Number within a given set
Name of the card
Rarity
Card Type
Element
HP Value
(multiple cards can be selected at once and stay selected until clicked again)
Each card cann be added through the "Add" button to an existing or new collection through the add button and then seen under thee collection tab of the website. 
The "Profile" button makes that specific card into the users profile picture and can be seen as such in the profile tab of the website.

Enjoy viewing and creating your own collections
